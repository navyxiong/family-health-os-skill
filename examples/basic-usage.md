3 个真实场景，展示从建档到生成摘要的完整流程。所有对话都是你 ↔ Hermes Agent。

## 场景 1: 老人慢病管理（高血压 + 房颤）

### 第一步：建档

**你说：**
> 给爷爷建档案。姓名王建国，男，1948年5月30日出生，湖南湘潭。既往高血压20年、房颤5年。过敏青霉素（皮疹）。父亲冠心病，母亲糖尿病。

**Agent 自动完成：**
- 写入 `memory/data/patients.json`
- 关联 `familyHistory: ["coronary_heart_disease", "diabetes"]`
- 触发聚合

**Agent 回复：**
```
已录入。患者档案 | 王建国 | 1事件
```

### 第二步：录入用药

**你说：**
> 他现在吃华法林 2.5mg 每天一次、厄贝沙坦 150mg 每天一次、胺碘酮 200mg 每天一次。


**你说：** （附一张凝血功能化验单图片）

2. 解析 INR=2.8（治疗范围），PT=32s

## 核心原则总结

| 原则 | 体现 |
|------|------|
| **零确认** | 所有录入静默完成，只输出 1 行 |
| **结构化** | 每次输出含表格/图标/来源标注 |
| **可溯源** | 每个事件标 `sourceDoctor` + `sourceInstitution` |
| **不越界** | 永远不做诊断建议，结尾固定 "*数据来源*" |
| **本地优先** | 所有数据 `cat memory/data/*.json` 直接可读 |
| **可扩展** | 7 个分析技能按需路由，无侵入 |
